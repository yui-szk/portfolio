# Portfolio Site

## Environments

- Next.js v14.2.4
- TypeScript v5.5.2
- Flowbite v2.3.0
- chart.js v4.4.3

## Getting Started

```sh
npm install
```

```sh
docker compose up
```

When use a hot reload

```sh
docker compose watch
```

```sh
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
